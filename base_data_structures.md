# Base structures

---

## Attributes

**Immutable**. Cannot be changed.
**Mutable**. Can change while the program is running.
**Heterogeneous**. Contains only one data type.
**Homogeneous**. Can contain multiple data types.

### Sequence data types 

Data types that allow for use of the following operations:

    - Slicing
    - Use of 'in' operator
    - Concatenation through + or *
    - The methods len, min, max, count, index
    - Comparisons of sequences (lexicographical comparison of corresponding 
    elements)
